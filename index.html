Python e HTML Juntos: Uma Dupla Poderosa na Web

Python e HTML são linguagens com propósitos diferentes, mas quando combinadas, tornam-se a base da maioria das aplicações web modernas. Pense no HTML como o esqueleto de um site e no Python como o cérebro que faz tudo funcionar por trás das cenas.

O que é HTML? (A Estrutura da Página)

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web.

Exemplo Simples de HTML:




   Minha Primeira Página


   
Bem-vindo ao meu site!

   
Este é um parágrafo de exemplo.

   Ir para o Google

é um título principal.

é um parágrafo. é um link.

O que é Python? (A Lógica por Trás da Cena)

Python é uma linguagem versátil usada no backend das aplicações web. Ela processa dados, interage com bancos e executa lógica de negócio.

Exemplo Simples de Python:

nome = "Mundo"
mensagem = f"Olá, {nome}!"
print(mensagem)

Como Eles Trabalham Juntos? (O Backend e o Frontend)

Python e HTML se conectam via frameworks como Flask ou Django. O Python processa dados e gera HTML dinâmico para o navegador.

  • Python lida com lógica e dados.
  • HTML exibe a interface para o usuário.

Exemplo Prático: Uma Página de Saudação Personalizada

Instale o Flask:

pip install Flask

app.py:

from flask import Flask, render_template_string
from datetime import datetime

app = Flask(__name__)

def get_current_time():
    return datetime.now().strftime("%H:%M:%S em %d/%m/%Y")

@app.route('/')
def home():
    nome_usuario = "Visitante"
    html_content = f"""
    
    
    
        Saudação Personalizada
    
    
        
Olá, {nome_usuario}!

        
Bem-vindo à sua página personalizada.

        
A data e hora atual é: {get_current_time()}

    
    
    """
    return render_template_string(html_content)

if __name__ == '__main__':
    app.run(debug=True)

Abra no navegador: http://127.0.0.1:5000/ para ver a página personalizada.

Python com Jinja2:

from flask import Flask, render_template
from datetime import datetime

app = Flask(__name__)

@app.route('/')
def home_template():
    nome_usuario = "Aventureiro"
    data_atual = datetime.now().strftime("%H:%M:%S em %d/%m/%Y")
    return render_template('index.html', nome=nome_usuario, data_hora=data_atual)

Resumo Final

  • HTML: Estrutura da página (o que você vê).
  • Python: Lógica e dados (o que acontece nos bastidores).
  • Combinados, eles criam aplicações web poderosas, dinâmicas e interativas.
